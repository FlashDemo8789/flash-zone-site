SVG Compression Tool

This tool will help compress the SVG while keeping the full design.

Original SVG (for reference)

Instructions:

  1. Open images/favicon-full.svg in a text editor
  2. Use an online SVG optimizer like SVGOMG
  3. Or use Nano SVG Compressor
  4. Save the compressed version as images/favicon.svg

Quick optimization tips:

  • Remove unnecessary metadata
  • Merge similar paths
  • Round decimal places to 2
  • Remove default attributes
  • Convert to path where possible
